  • Start filling in the details as asked. Once you have selected the Model for which you want to make the dashboard item, you will be asked to select the Dashboard Item Type. There are eight dashboard item types being provided in Dashboard Ninja Pro: Tiles, Bar Chart, Line Chart, Area Chart, Pie Chart, Doughnut Chart, Polar Area Chart, and List View.
  • If you select the Dashboard Item Type as a tile, you will see that the model field is mandatory in order to save the dashboard item. Once the Model is selected, you will see a Date Filter Field appear. The Date Filter Field will have all the date fields available in the selected model as a drop-down menu.

    For Example: In the screenshot, you can see the date filter applied for Last 7 Days.

    For Example: If your selected Model is Sale Order, you can define the Domain as Sale Order where the total amount is greater than 100.
  • Now, for the defined Domain, you have further options of calculating the Record Count, calculate its Sum, and its average depending on your need.
  • Select the Theme as per your choice. If you wish to choose your own color, you can do the same through Background Color. You can even change the Transparency of the chosen color.
  • Change the Font Color of the dashboard item and adjust its Transparency as per your choice.
  • Choose the icon for your dashboard item. If you wish to choose from our Default Icons, choose the Default Icons option and if you wish to upload your own, choose Upload Icon option.
  • If you have chosen the Default Icons option, the below pop-up window will be displayed. Choose the icon you like and if you wish to explore more icons, search them in the search bar.
  • Finally, choose the Icon Color for your dashboard item and adjust the Transparency accordingly.
  • See the Preview of your finished dashboard item. If you like what you see, proceed to save the same.
    Note: Only users who have access to make dashboards in your Dashboard Ninja Settings, will be able to add the dashboard item and customize the dashboards.
  • If you select the Dashboard Item Type as a Bar Chart, you will see three mandatory fields have been displayed that need to be filled in order to save the dashboard item.
  • Once you have selected the Dashboard Item Type as a Bar Chart, you will then need to select the Chart Data Count Type. You have the option of selecting either Count or Sum for the given model.
  • The next step involves choosing the Measures for the given model.
    For example: Let’s select the Model as Sales Order and keep its Measures as Total (sale order).
  • Once the Measures is selected, you then need to select the Group By for which the given Measures need to be grouped.
    For example: For the Model and Measures taken in the step before this, let’s select the Group By as Customer (sale order).
  • You can also select the Sort By Field, Sort Order, and Record Limit option specific to the dashboard item you’re currently making.

The same can also be defined from the main dashboard by clicking at the brush icon named Color Palette on each individual dashboard item whose type is charts.

  • Once you have saved the dashboard item, you will see an Action tab appear. Clicking the same will reveal two drop-down options to either Duplicate the dashboard item or Move the dashboard item.
  • Selecting either of the options will open the Dashboard Item Action pop-up. Proceed by selecting the Action you want to execute i.e. Duplicate and Move, and the destination Dashboard.
  • The Move Action will move the dashboard item from one dashboard to another dashboard.
  • The Duplicate Action will duplicate the dashboard item from one dashboard to another or the same dashboard.
  • You can execute the Duplicate and Move Action from your dashboard as well as each respective dashboard item has an Action icon.
  • You can customize as many dashboards you want and the same will be maintained in the Dashboard Manager.
  • You can activate and deactivate a customized dashboard by checking and unchecking the box respectively.

Export & Import complete dashboard between different databases
  • Steps to export dashboard
    1. Login with admin account
    2. Go to menu : My Dashboard > Dashboard Manager
    3. Select the dashboard record which you want to export
    4. Select "Export Dashboard" option from 'Actions' dropdown 5. A JSON file will be downloaded with name 'dashboard_ninja.json'.
  • Steps to import dashboard
    1. Login with admin account
    2. Go to menu : My Dashboard > Dashboard Manager
    3. Click on 'Import Dashboard' button
    4. Select the JSON (dashboard_ninja.json) file exported previously 5. Dashboard will be imported successfully and you can see in list view of Dashboard Manager
Use of %UID in domain filter to filter logged in user data
  • If you want to see data of logged In user for particular item, follow below steps:
    1. Go to edit screen for particular dashboard item
    2. Click on 'Domain Filter'
    3. In value field, use %UID
    4. Click Save
Installation Steps for Existing Customers
  • Existing Dashboard Ninja Pro and Dashboard Ninja Filters customers are requested to uninstall the aforementioned apps except Dashboard Ninja from their system and then upgrade to this merged Dashboard Ninja app.
  • If you upgrade to the current Dashboard Ninja app without uninstalling Dashboard Ninja Pro and/or Dashboard Ninja Filters, data conflict issues might arise. If you have any query regarding data migration, please contact sales@ksolves.com for help.
  • Once Dashboard Ninja Pro customers will uninstall the app, their charts dashboard items will get converted to normal tiles. They can then restore the same by installing the new Dashboard Ninja app.
  • After upgrading the Dashboard Ninja app, existing users will need to execute the following steps to restore their earlier data:
    1. Go to Dashboard Manager.
    2. Select all dashboard records.
    3. Click on the Action tab displayed in the middle top section of the window.
    4. After that, select "Recover Previous Data" option to recover all dashboard items created in earlier versions.

    When you create a dashboard, an action is created with name : Menu Name + "Action". So for example if you create a dashboard with name : Project Dashboard. Go to any user page -> Preference -> Home Action and can select action with name : Project Dashboard Action.
  • I have mistakenly created a chart item on Sales Dashboard, instead of recreating it again, Can I move it or duplicate?
    Yes, we have provided a feature to move and duplicate any item from one dashboard to another. When you mouseover any item you will see a copy icon or click on edit icon and from top center of screen there is Actions dropdown with options to move/duplicate items.
  • How to make a bar chart full size on the screen?
    Pretty simple, just click on Gear like icon present at top right of screen and then click on edit layout option. Now, when you mouseover any tile or chart you will see resize icon at bottom right of item using which you can resize items as per your need.
